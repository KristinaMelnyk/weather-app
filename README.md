# Weather App

A clean, glassmorphism-styled weather app built with vanilla HTML, CSS, and JavaScript.

## Features

- Search any city in the world
- Live temperature, humidity, wind speed, and feels-like temperature
- Weather condition icons and descriptions (clear, cloudy, rain, snow, etc.)
- Glassmorphism UI with animated background blobs
- Error handling for invalid city names
- Press Enter or click Search to look up weather

## Built with

- HTML, CSS, JavaScript (no frameworks)
- [Open-Meteo API](https://open-meteo.com/) — free, no API key required
